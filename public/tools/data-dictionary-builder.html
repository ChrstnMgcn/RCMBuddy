Build Custom Data Dictionaries

Define validation rules for your Excel file headers and save them as reusable data dictionaries for your "Excel Validate" tool.

How it works:

Upload an Excel or CSV file to extract its headers. For each header, you can then specify various validation rules, such as whether a field is required, what values are allowed, numeric ranges, date rules, or even custom regular expressions. Once defined, give your data dictionary a name and save it. It will then be available for use in the "Excel Validate" tool to ensure data quality across your spreadsheets.

  • **Upload File:** Select an Excel/CSV file to load its column headers.
  • **Define Rules:** For each header, choose a validation type and provide a corresponding value/message.
  • **Save Dictionary:** Give your dictionary a name and save it to your account.
  • **Use in Validation Tool:** Your saved dictionaries will appear in the "Excel Validate" tool for quick application.
